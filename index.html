To use MAJAX 2, include in your webpage this: 

<script type="text/javascript">
// You are free to use our service for testing, but there is no
// guarantee it will be always available
var majax2ServiceUrl = "http://libx.lib.vt.edu/services/majax2/";

// Point this at your catalog.  
// It should work with any III system.
var majax2OpacBase = "http://addison.vt.edu/search";
</script>
<script type="text/javascript" src="majax2.js"></script>

"MAJAX searchterm" goes into the title attribute.
"MAJAX class" goes into the class attribute. Order matters.
"MAJAX output" is the output produced
